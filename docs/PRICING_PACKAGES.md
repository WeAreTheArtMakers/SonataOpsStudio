# Pricing Packages

All ranges are indicative in USD and assume one production environment.

## Pilot (2 weeks)
- **$18k - $35k** one-time
- Includes setup, one domain integration, 1 workflow, 1 executive brief template
- Assumption: customer provides data feed and 10 seed docs

## Standard (6 weeks)
- **$65k - $140k** one-time + optional support
- Includes 5 KPI domains, full RAG onboarding, 3 workflows, eval playbook
- Assumption: single business unit, no custom SSO required

## Enterprise (12 weeks)
- **$220k - $480k** implementation + annual support
- Includes SSO/RBAC, governance extensions, multi-tenant setup, SLA package
- Assumption: 2+ environments, procurement and security reviews included

## Optional Annual Support
- Silver: $30k/year (business-hours)
- Gold: $90k/year (extended support + quarterly optimization)
- Platinum: custom (24x7 + dedicated TAM)

## Pricing Drivers
- number of KPI domains
- doc corpus size and refresh frequency
- automation complexity and integrations
- governance and compliance requirements
