# Demo Script (12-15 min)

## 1. Hook (1 min)
"Most ops platforms stop at charts. SonataOps adds hearing + grounded AI reasoning + automated executive narration."

## 2. KPI + Anomaly (3 min)
1. Open Dashboard.
2. Show live KPI cards and anomaly trend.
3. Open Anomalies page, filter by `RiskScore`, open detail drawer.

## 3. Explain with Grounded Copilot (3 min)
1. Click **Explain this anomaly**.
2. Show answer with source citations.
3. Mention PromptOps approval workflow and tenant-scoped retrieval.

## 4. Listen to the Data (3 min)
1. Open Listen page.
2. Select metric window + preset `Risk Tension`.
3. Generate render, wait for completion event.
4. Play signed URL asset in browser.

## 5. Automation Story (2 min)
1. Show n8n flow exports.
2. Trigger manual brief generation.
3. Open Briefs page and preview markdown output.

## 6. Enterprise Close (2 min)
- OTel trace in Jaeger
- Grafana metrics dashboard
- governance notes: redaction, audit logs, prompt approval
- rollout options from pricing/implementation docs
